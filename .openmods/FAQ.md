### How do I install this mod?
Download the latest release from the sidebar and place the contents of the zip file into your game's `BepInEx/plugins` folder. 
Make sure you have the following prerequisites installed:
- **BepInEx 6.0** or higher
- **Nautilus** (for Subnautica mods)

### Is this compatible with multiplayer?
Yes! This mod is designed to work seamlessly in multiplayer sessions. However, **all players** must have the mod installed to avoid sync issues.

### Can I include this in my modpack?
Absolutely. We encourage modpack creators to include our work. A credit link back to this OpenMods page is appreciated but not required.

### I found a bug! Where can I report it?
Please use the **"Report a Bug"** button in the Links section to open an issue on our GitHub repository. 
When reporting, please include:
1. Your game version
2. A description of the issue
3. Your `LogOutput.log` file from the BepInEx folder.

### Does this mod work on Linux or macOS?
Yes, as long as you are using the appropriate version of BepInEx for your platform, the mod should function correctly.
